## About the Project

Praxe E-Shop is a fictional online store built as part of a school practice (Praxe) assignment.
The goal is to learn modern React development practices: component architecture, state management, routing, and consuming a REST API.

**Features:**
- Home page with hero section, featured products and benefits
- Product catalog with category filter, search and sorting
- Individual product pages with reviews
- Shopping cart (Context API)
- Contact form
